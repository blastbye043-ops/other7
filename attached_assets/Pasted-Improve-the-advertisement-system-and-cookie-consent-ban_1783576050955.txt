ADVERTISEMENT REFRESH
==================================================

Do NOT implement any refresh behavior that violates Google AdSense policies.

Instead:

• Create an ad refresh manager that refreshes only where appropriate and policy-compliant.

• Only refresh ads when:
  - The user has been active on the page.
  - The ad slot is visible.
  - The refresh interval has elapsed.
  - Refresh behavior complies with Google AdSense requirements.

• Never refresh hidden ads.

• Never refresh ads while the tab is inactive.

• Pause refresh when the browser tab loses focus.

• Resume refresh when the tab becomes active again.

• Use IntersectionObserver for visibility detection.

• Prevent duplicate ad requests.

• Destroy previous ad instances before requesting a new one if required.

• Avoid unnecessary network requests.

• Do not refresh every advertisement simultaneously.

• Randomize refresh timing slightly to reduce simultaneous requests.
